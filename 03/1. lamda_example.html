* 화살표 함수

Javascript ES6에서 새롭게 등장한 화살표 함수.
  • function func_name() { }
  • function func_name(a, b) { }
  • function func_name() { return value; }
  • function func_name(a, b) { return value; }
이를 화살표 함수로 나타내면 다음과 같다
  • () => { }
  • (a, b) => { }
  • () => { return value; }
  • (a, b) => { return value; }
